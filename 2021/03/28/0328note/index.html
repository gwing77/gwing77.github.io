vue 命令工具安装

npm show vue-cli 查看版本
npm install -g vue-cli@2.9.6 指定版本，全局安装
vue -V 查看版本
vue -h 查看命令
Commands:
init generate a new project from a template//$ vue init webpack my-project
list list available official templates
build prototype a new project
create (for v3 warning only)
help [cmd] display help for [cmd]

vue 的项目模板

Available official templates:

★ browserify - A full-featured Browserify + vueify setup with hot-reload, linting & unit testing.
★ browserify-simple - A simple Browserify + vueify setup for quick prototyping.
★ pwa - PWA template for vue-cli based on the webpack template
★ simple - The simplest possible Vue setup in a single HTML file
★ webpack - A full-featured Webpack + vue-loader setup with hot reload, linting, testing & css extraction.
★ webpack-simple - A simple Webpack + vue-loader setup for quick prototyping.

常用的命令

  • npm run dev 运行web-server
  • npm run build 项目打包，之后才能发布，部署给用户
  • npm run unit 单元测试

    项目目录

    src:项目源文件，基于此开发
    dist:打包文件夹
    package.json:项目设置文件

    引入bootstrap框架

    npm install bootstrap –save –save-exact//存在package.json中，版本号不带^

添加自己的CSS文件

assets下建立mycss.css，在App.vue的script中引用，之后就能用了
App.vue 是整个网页的入口，入口的js是main.js,入口的组件叫App.vue

组件结构

vue命令行给我们带来了组件开发方式
所有的组件都放在工程的组件文件夹中src/components
组件格式包含三个部分

  • template 组件的html内容
  • script 组件的js，要用ES6的语法
  • style 组件的css 加scoped只在本组件生效

    webpack模板生成的工程结构

  • build
    • webpack的设置文件
  • config
    • 工程设置文件
  • src
    • 源文件夹
  • static
    • 静态文件夹（编译的时候会直接拷贝到发布文件夹）
  • index.html
    • 网页启动入口文件，我们开发的是单页应用程序，所有的组件都是在这个页面的div里面进行渲染的，可以修改文件头部

      路由vue-router的设定

      npm install vue-router –save –save-exact
  • src/router/index.js
    • 引入路由组件
    • 将Router注册到全局区
    • 输出一个Router类，定义了各种各样的url映射，将组件绑定给url
  • src/main.js
    • import router
    • 将router在入口组件中注册，这样所有的子组件都能访问到路由的属性

      嵌套路由

      在动态路由的基础上再加上嵌套的URL

      对路由进行编程

      除了router-link的方式，也可通过编程来实现
  • router.push(location, onComplete?, onAbort?)
  • this.$router.push({path:/player/${uid}});
  • this.$router.push({name: ‘Player’ , params : {uid : uid}});
  • this.$router.push({path: ‘/Player’ , query : {uid : uid}});//通过get方式传参
  • this.$router.go(-1);
